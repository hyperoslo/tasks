Tasuku

| tɑːsuku |

Tasuku is a framework for building tasks in Ruby on Rails.

Tasuku ships with an array of tasks ranging from reading articles to sharing photos. You can make your own tasks, too, and we'd love for you to contribute if you do.

Best of all, the code is already written for you. All you have to do is make it look nice.
