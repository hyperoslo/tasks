## Installation

Add this line to your application's Gemfile:

    gem 'tasks'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tasks

Install the migrations:

    rake tasks:install:migrations
    
Run the migrations:

    rake db:migrate
